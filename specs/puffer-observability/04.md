# Telegram Import Alias Removal

- `TelegramImportLocal` is no longer listed in the default always-redacted
  tool id set because the tool id has been removed.
- `TelegramImportDesktop`, `TelegramLoginQr`, and `TelegramLoginQrWait` remain
  always-redacted to avoid leaking auth paths, QR login URLs, or local
  passcodes.
