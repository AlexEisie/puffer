# puffer-observability update 02

Extends Telegram secret redaction for local auth import.

Design and behavior:

- `TelegramImportLocal` is included in the default always-redacted tool id
  list.
- The broader `Telegram` internal tool id remains always redacted, covering
  passcodes when import is invoked through the CLI helper.

Compatibility:

- Existing login-code and 2FA redaction behavior is unchanged.
