# Slack tool and skill resources

## Summary

Bundled resources now include a Slack internal tool schema and Slack skill
guidance.

## Design

- `resources/internal_tools/slack.yaml` declares the hidden `Slack` runtime
  handler, auth/import/lookup actions, file read/write metadata, and strict
  input schema.
- `resources/skills/slack/SKILL.md` tells agents to use Bash `slack ...`,
  distinguish `slack-app` from `slack-login`, resolve Slack ids before acting,
  and use `ConnectorAct` for sends/uploads/reactions.
- Connector and connection list descriptions include `slack-app` and
  `slack-login` examples.

## Compatibility

No model-visible tool surface is added. The resources document the new Slack
connector workflow while preserving the existing connector tools.
