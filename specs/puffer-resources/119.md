# Canvas Offer AskUserQuestion Metadata

## Design

`Canvas` instructs agents to offer long structured answers through
`AskUserQuestion` with `metadata.kind = "canvas-offer"`. The
`AskUserQuestion` resource schema now explicitly allows that `kind` string while
keeping metadata closed to unbounded keys.

## Logic

The schema remains restrictive (`additionalProperties: false`) and accepts only
the existing `source` marker plus the new `kind` marker. Desktop clients can
continue detecting the Canvas offer affordance from the normalized question
metadata without requiring a separate tool or prompt shape.

## Compatibility

Existing `AskUserQuestion` calls are unchanged. The update only makes the
documented Canvas offer contract valid for provider-side tool schema validation.
