# Searchable AskUserQuestion Choices

`AskUserQuestion` resources now expose a searchable choice mode for bounded
catalog selection.

- Choice questions may set `searchable: true` to request a filterable
  single-select list.
- Searchable questions may provide up to 50 options; ordinary choice questions
  still describe the compact 2-4 option decision shape.
- `searchable` is documented as choice-only and incompatible with
  `multiSelect`.
- This supports `/connect` connector selection without falling back to
  free-form slug entry or provider-prompt interpretation.
