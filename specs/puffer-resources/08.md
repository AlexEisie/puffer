# Telegram Resource Naming

- `resources/internal_tools/telegram.yaml` exposes `import_desktop` as the
  preferred Telegram Desktop `tdata` import action and keeps `import_local`
  only for compatibility.
- The Telegram internal tool schema includes `login_qr`, `login_qr_wait`, and
  `timeout_seconds`.
- The Telegram skill now recommends QR login first for users with any logged-in
  Telegram app, including native macOS Telegram.
- The skill documents Telegram Desktop import separately and states that
  `import-local` is not native macOS Telegram import.
