# Vercel Webhook Skill

## Scope

Adds a bundled `vercel-webhook` skill that documents deterministic setup and
runtime behavior for the Vercel webhook preset.

## Design

- The skill routes users to `/connect vercel-webhook <connection-name>` with
  the deterministic default `/connect vercel-webhook vercel-webhook`.
- It documents that the preset writes the generic `[connectors.webhook]`
  configuration and is served by `puffer serve`.
- It lists the supported Vercel event families and the conversation grouping
  keys used by the webhook normalizer.

## Compatibility

The skill is additive and follows the same resource model as the other webhook
connector skills.
