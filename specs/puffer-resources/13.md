# Telegram Message Search Media Documentation

- The Telegram skill now documents message-search media values.
- Downloadable media is documented as a local file path, and semantic media
  such as polls is documented as text.
- The Telegram internal tool schema describes succinct output as plain text
  that can include media paths or semantic media text.
- JSON output remains available for callers that need structured message
  fields, but its `media` value follows the same path-or-text contract.
