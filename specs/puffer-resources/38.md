# Asana Webhook Skill Resource

## Summary

Adds the bundled `asana-webhook` skill so agents and users can discover the
deterministic setup path for Asana webhook deliveries.

## Skill Contract

The skill documents:

- `/connect asana-webhook <connection-name>`
- deterministic default `/connect asana-webhook asana-webhook`
- default serve path `/asana`
- `[connectors.webhook]` storage because the preset is served by the generic
  webhook connector
- `puffer serve` as the runtime command
- Asana handshake, signature, heartbeat, and event-array behavior

## Compatibility

Bundled skill count increases by one. The tmux help wide snapshot is updated so
resource counts remain deterministic.
