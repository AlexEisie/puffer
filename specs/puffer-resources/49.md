# Azure DevOps Webhook Skill

Bundled resources now include the `azure-devops-webhook` skill for
deterministic connector setup guidance.

## Contract

- The skill teaches `/connect azure-devops-webhook <connection-name>` and the
  deterministic default
  `/connect azure-devops-webhook azure-devops-webhook`.
- Setup guidance explains that the preset is backed by `[connectors.webhook]`
  and started with `puffer serve`.
- Azure DevOps configuration guidance covers Service Hooks, Web Hooks, code
  pushed events, pull request events, pull request comments, and work item
  events.
- Runtime guidance documents the stable Azure DevOps conversation id forms.

## Compatibility

The resource does not add outbound connector actions or a subscriber runtime.
It only makes the existing declarative skill catalog aware of the new
serve-mode preset.
