# Telegram Message Search Resources

- The bundled Telegram internal tool schema now includes the `search_messages`
  action.
- The schema accepts `peer`, `query`, `limit`, and `context` fields for
  message search while preserving the existing login, import, and peer lookup
  actions.
- The bundled Telegram skill now instructs agents to resolve a peer id first,
  then run `telegram search-messages <query> --peer <id> --context N`.
