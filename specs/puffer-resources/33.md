# GitHub Webhook Skill Resource

## Summary

Bundled resources include a `github-webhook` skill for GitHub webhook setup
guidance.

## Behavior

- The skill documents `/connect github-webhook <connection>` as the setup
  command.
- It explains that the connector is a GitHub-specific preset over the generic
  HTTP webhook transport.
- It documents the workspace config shape under `[connectors.webhook]` and the
  need to run `puffer serve`.
- It states that the connector is setup-only until webhooks expose durable
  workflow trigger events.

## Compatibility

The new skill is additive. Existing skills, resource provenance loading, and
connector setup resources keep their prior behavior.
