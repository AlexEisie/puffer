# Datadog Webhook Skill Resource

## Change

Bundled resources now include `resources/skills/datadog-webhook/SKILL.md`.

## Skill Contract

The skill documents:

- deterministic setup through `/connect datadog-webhook <connection-name>`
- default command `/connect datadog-webhook datadog-webhook`
- default serve path `/datadog`
- `.puffer/connectors.toml` writing through `[connectors.webhook]`
- `puffer serve` as the runtime entrypoint
- Datadog Webhooks integration setup with `@webhook-<WEBHOOK_NAME>`
- a Puffer-friendly Datadog custom JSON payload built from documented Datadog
  webhook variables
- conversation routing and message summary behavior

## Compatibility

The bundled skill count increases by one. Tmux help snapshots and resource
loader tests must continue to parse all bundled skill markdown.
