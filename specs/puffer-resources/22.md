# Slack Skill Capability Notes

## Scope

Updates the Slack skill text to match runtime capabilities.

## Guidance

The skill now describes `slack-app` as an app install and bot-token action
connector, and states that Slack workflow subscriptions and agent proxy mode are
not implemented until a typed Slack subscribe runtime exists.

## Compatibility

Agents should continue using Slack lookup commands and `ConnectorAct` for
outbound messages, uploads, and reactions. They should not create Slack
workflows that depend on inbound Slack events yet.
