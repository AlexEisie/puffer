# Grafana Webhook Skill Resource

## Change

Bundled resources now include `resources/skills/grafana-webhook/SKILL.md`.

## Skill Contract

The skill documents:

- deterministic setup through `/connect grafana-webhook <connection-name>`
- default command `/connect grafana-webhook grafana-webhook`
- default serve path `/grafana`
- `.puffer/connectors.toml` writing through `[connectors.webhook]`
- `puffer serve` as the runtime entrypoint
- Grafana Alerting webhook contact point setup notes
- current lack of `X-Grafana-Alerting-Signature` verification

## Compatibility

The bundled skill count increases by one. Tmux help snapshots and resource
loader tests must continue to parse all bundled skill markdown.
