# RelayDance Video Catalog Expansion

Date: 2026-06-09

## Summary

The RelayDance provider resource now declares the verified video models returned
by RelayDance `/v1/models` that fit the existing text-to-video adapter boundary.

## Behavior

- RelayDance video catalog expands from 3 to 12 selectable models.
- Added Seedance 1.5, Seedance NSFW, Grok Imagine Video, and HappyHorse text-to-video entries.
- HappyHorse image/reference/video-edit models remain excluded because the
  current adapter does not accept media-reference inputs.
- Prompt-only models stay parameterless until RelayDance exposes structured
  parameter metadata for them.

## Compatibility

No runtime discovery is added. The video settings surface still reads static
`media.video.models` descriptors through the existing capability resolver.
