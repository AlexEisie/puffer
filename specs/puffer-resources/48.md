# Bitbucket Webhook Skill

Bundled resources now include the `bitbucket-webhook` skill for deterministic
connector setup guidance.

## Contract

- The skill teaches `/connect bitbucket-webhook <connection-name>` and the
  deterministic default `/connect bitbucket-webhook bitbucket-webhook`.
- Setup guidance explains that the preset is backed by `[connectors.webhook]`
  and started with `puffer serve`.
- Bitbucket configuration guidance covers repository and workspace webhook URLs,
  `X-Event-Key`, repository push events, pull request lifecycle events, pull
  request comments, and issue/comment events.
- Runtime guidance documents the stable Bitbucket conversation id forms.

## Compatibility

The resource does not add outbound connector actions or a subscriber runtime.
It only makes the existing declarative skill catalog aware of the new
serve-mode preset.
