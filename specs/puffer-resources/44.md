# Alertmanager Webhook Skill Resource

## Change

Bundled resources now include `resources/skills/alertmanager-webhook/SKILL.md`.

## Skill Contract

The skill documents:

- deterministic setup through `/connect alertmanager-webhook <connection-name>`
- default command `/connect alertmanager-webhook alertmanager-webhook`
- default serve path `/alertmanager`
- `.puffer/connectors.toml` writing through `[connectors.webhook]`
- `puffer serve` as the runtime entrypoint
- Prometheus Alertmanager `webhook_config` setup notes
- the default Alertmanager webhook JSON fields used for routing and summaries

## Compatibility

The bundled skill count increases by one. Tmux help snapshots and resource
loader tests must continue to parse all bundled skill markdown.
