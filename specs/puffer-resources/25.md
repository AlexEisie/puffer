# 25 - Slack Skill Action-Only Wording

The Slack skill now describes the internal CLI as connector/account setup and
lookup, not as a workflow trigger surface.

- The existing warning that Slack workflow subscriptions and agent proxy mode
  are not implemented remains in place.
- Agents should use Slack lookup commands to resolve ids and `ConnectorAct` for
  outbound side effects.

This avoids implying inbound Slack workflow support before a typed Slack
subscribe runtime exists.
