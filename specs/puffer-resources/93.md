# VideoGeneration Tool Resource

## Scope

Bundled resources now include a model-facing workflow tool definition for
agent text-to-video generation.

## Change

- Added `resources/tools/video_generation.yaml`.
- The tool id and name are `VideoGeneration`.
- The handler is `runtime:workflow:video_generation`.
- The tool uses `approval_policy: ask` and `sandbox_policy: network`.
- The input schema requires `prompt`, accepts optional string-valued scalar
  `parameters`, accepts optional `purpose`, and sets
  `additionalProperties: false`.
- The description states that the tool is text-to-video only and produces one
  persisted video artifact for one logical request.

## Compatibility

The schema intentionally does not expose image, reference image, first-frame,
last-frame, URL, count, seed, scheduler, or provider-specific fields.
