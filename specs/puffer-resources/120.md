# Canvas Desktop Inline Wording

## Summary

The Canvas resource now tells agents that Puffer Desktop also renders Canvas
inline inside the chat, while the generated browser page remains the fallback
for non-inline hosts.

## Contract

- Canvas specs and component types are unchanged.
- `.puffer/canvas/<canvasId>.state.json` remains the source of truth for
  interactive values.
- The generated page remains available for non-desktop/default browser fallback.
