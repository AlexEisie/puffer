# BytePlus Provider Resource

`resources/providers/bytepluses.yaml` registers BytePlus Ark as an
OpenAI-compatible chat-completions provider.

- The provider id is `bytepluses` and the base URL is
  `https://ark.ap-southeast.bytepluses.com/api/v3`.
- `chat_completions_path: /chat/completions` prevents the OpenAI completions
  client from appending the default `/v1/chat/completions` suffix to an
  already-versioned Ark API URL.
- API-key auth uses the normal bearer-token provider path.
- Runtime model discovery uses `/models` with the OpenAI model-list response
  shape, so deployed Ark model ids can be selected without hardcoded fallback
  models.
