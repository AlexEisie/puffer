# BytePlus Seedream Sequential Image Parameter

## Scope

BytePlus native Seedream image descriptors should expose the provider parameter
needed for multi-image generation.

## Change

- `seedream-5-0-260128`, `seedream-4-5-251128`, and `seedream-4-0-250828`
  now declare `sequential_image_generation`.
- The parameter values are `disabled` and `auto`, with `disabled` as the static
  descriptor default.
- The governance test now treats this parameter as part of the native Seedream
  image API surface.

## Compatibility

Single-image requests remain descriptor-default `disabled`. Runtime count-aware
handling switches the parameter to `auto` only when more than one image is
requested.
