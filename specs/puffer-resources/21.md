# Lark Resources

## Scope

Adds declarative resources for the Lark internal tool and skill.

## Resources

- `resources/internal_tools/lark.yaml` defines the internal `Lark` workflow
  tool schema and aliases.
- `resources/skills/lark/SKILL.md` teaches agents to authenticate Lark app or
  user-token connections, resolve ids before acting, read/search messages, and
  use `ConnectorAct` for side effects.
- Connector and connection list descriptions now mention Lark templates.

## Compatibility

The skill requires id resolution before sending to named users/chats and records
that `lark-app`/`lark-login` do not support subscription or proxy workflows yet.
