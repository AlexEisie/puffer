# puffer-resources update 07

Documents Telegram local auth import in resources.

Design and behavior:

- The Telegram internal tool schema now includes `import_local` plus optional
  `path`, `account_index`, `passcode`, and `key_file` fields.
- The Telegram skill now tells agents to try `telegram import-local` before
  falling back to phone/code login when the user wants to reuse a local
  Telegram Desktop account.
- The internal tool metadata now marks Telegram as reading files because
  import reads the Telegram Desktop `tdata` directory.

Security:

- The skill treats local passcodes as secrets and prefers stdin-based input.
