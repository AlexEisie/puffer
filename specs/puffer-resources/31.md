# Serve Connector Skills

## Summary

Bundled skills now describe the Discord, Matrix, and webhook `puffer serve`
connector setup paths.

## Behavior

- `discord`, `matrix`, and `webhook` skills point users to connector TOML
  configuration and `puffer serve`.
- Each skill explicitly states that the connector is not yet a workflow trigger
  and should not be used for `WorkflowCreate` connection triggers until a typed
  subscriber or connector protocol stream exists.

## Compatibility

The skills are additive. Existing Telegram, Slack, Lark, and email connector
skills are unchanged.
