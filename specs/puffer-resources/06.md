# puffer-resources update 06

Model-visible tool resources now expose connector, connection, and workflow
create terminology.

Added resources:

- `ConnectorCreation`
- `ConnectorList`
- `ConnectorTemplate`
- `ConnectorRegister`
- `ConnectorUpdate`
- `ConnectorDelete`
- `ConnectorAct`
- `ConnectionCreate`
- `ConnectionList`
- `ConnectionDelete`
- `WorkflowCreate`
- `WorkflowList`
- `WorkflowToggle`
- `WorkflowValidate`
- `WorkflowInspect`

Removed resources:

- `SubscriberInstall`
- `SubscriberList`
- `SubscriberScaffold`
- `SubscriptionCreate`
- `SubscriptionList`
- `SubscriptionPause`
- `SubscriptionDelete`
- legacy workflow registration YAML resource

`WorkflowCreate` is the only model-visible workflow write resource. It accepts
`slug`, `connection_slug`, optional filter/classifier fields, and either
`action` JSON or `yaml_action`.
