# Lark Thread Read Guidance

## Scope

Updates Lark internal tool metadata and skill guidance for thread reads.

## Behavior

- `resources/internal_tools/lark.yaml` documents `thread_id` for
  `read_messages`.
- The Lark skill tells agents to use `--thread-id` when they need replies from
  a specific Lark thread.

## Compatibility

No existing tool names or arguments are removed. This only makes the Lark read
surface match the runtime's new thread-container support.
