# Trello Webhook Skill Resource

## Summary

Adds the bundled `trello-webhook` skill so agents and users can discover the
deterministic setup path for Trello webhook deliveries.

## Skill Contract

The skill documents:

- `/connect trello-webhook <connection-name>`
- deterministic default `/connect trello-webhook trello-webhook`
- default serve path `/trello`
- `[connectors.webhook]` storage because the preset is served by the generic
  webhook connector
- `puffer serve` as the runtime command
- Trello callback validation, `X-Trello-Webhook`, and action/model payload
  behavior

## Compatibility

Bundled skill count increases by one. The tmux help wide snapshot is updated so
resource counts remain deterministic.
