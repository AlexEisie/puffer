# Jira Webhook Skill Resource

## Design

The bundled `jira-webhook` skill documents the deterministic
`/connect jira-webhook <connection>` setup flow, the default `/jira` path, and
the `puffer serve` runtime requirement.

## Contracts

- The skill explains that Jira webhooks should post to the configured public
  URL.
- It lists the recognized Jira event families: issue, comment, and project
  events.
- It documents setup-only catalog behavior so agents do not claim a workflow
  trigger runtime or outbound action support for this preset.

## Compatibility

The new skill only increases bundled skill inventory. Existing skill names and
aliases remain unchanged.
