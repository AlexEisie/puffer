# WorkflowCreate planned connection schema

`resources/tools/workflow_create.yaml` documents planned workflow creation.

## Schema

- `connector_slug` is optional and may be supplied when `connection_slug` does
  not exist yet.
- Callers may omit `connector_slug` for built-in suggested connection names such
  as `telegram-user` and `email`; runtime inference resolves those names to a
  trigger-capable connector.
- `connection_slug` now describes both authenticated connections and planned
  connection slugs.

The tool schema remains a top-level object without `oneOf`, `anyOf`, `allOf`,
`enum`, or other provider-hostile top-level constructs.
