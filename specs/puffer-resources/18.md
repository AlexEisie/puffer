# Telegram connector action guidance

- The Telegram skill now distinguishes internal CLI operations
  (login/import/peer lookup/message search) from connector actions
  (`ConnectorAct` on `telegram-login`).
- The skill lists the supported Telegram action slugs for messages, polls,
  reactions, chat state, membership, moderation, profile, group metadata,
  avatars, and stories.
- The internal Telegram tool metadata states that sends, poll votes, group
  updates, profile updates, reactions, moderation, and stories are connector
  actions rather than internal CLI actions.
