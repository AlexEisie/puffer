# Slack internal CLI descriptor

## Summary

The shared internal CLI tool descriptor list now includes Slack.

## Design

- The `slack` alias expands to `puffer internal-tool slack "$@"`.
- The descriptor points agents at the `slack` skill resource.
- Slack is registered as an internal tool resource only, never as a
  model-visible provider tool.

## Compatibility

Browser, Email, and Telegram helper generation is unchanged. Existing tests now
assert Slack remains hidden from the normal tool registry and present in the
internal registry.
