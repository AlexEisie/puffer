# Lark Internal Tool Descriptor

## Scope

Adds `lark` to the first-party CLI-only internal tool descriptor list.

## Behavior

- Shell helpers now include `lark() { puffer internal-tool lark "$@"; }`.
- The descriptor maps to skill resource `lark`.
- `Lark` is registered as an internal-only tool resource and remains hidden
  from the normal provider tool surface.

## Compatibility

The descriptor follows the existing browser/email/slack/telegram internal tool
pattern and does not alter public tool registration semantics.
