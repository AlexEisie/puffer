# Workflow Connection Repair Commands

`/workflows connections` now shows the deterministic repair command for each
stored connection.

Behavior:

- Connection rows include `repair=/connect <connector_slug> <connection_slug>`.
- Query matching includes the repair command and the word `repair`, so users
  can search for commands such as `/workflows connections repair telegram`.
- Trigger-ready connections still include `monitor=/monitor <connection_slug>`.
- Non-trigger connections still omit monitor commands but keep repair commands.

Compatibility:

- The command output format remains plain text and additive.
- Existing connection slug, connector slug, description, and monitor search
  behavior is preserved.
