# Telegram Import Alias Removal

- The consolidated internal `Telegram` workflow tool accepts
  `action=import_desktop` for Telegram Desktop `tdata` import.
- `action=import_local` and the direct `TelegramImportLocal` workflow handler
  are removed to keep the auth surface unambiguous.
- Permission summaries and internal secret redaction now special-case only
  `import_desktop`; QR and phone-code login handling is unchanged.
