# Telegram Message Search Internal Action

- The consolidated `Telegram` internal tool now handles `search_messages`.
- The action validates non-empty `peer` and `query`, ensures the
  `telegram-user` subscriber is running, sends `TelegramSearchMessages`, and
  waits for `message_search` or `message_search_error`.
- Successful responses preserve the subscriber payload and tell the agent to
  inspect `results[].context`, where the matched item has `is_match: true`.
- Permission summaries now identify `search_messages` as Telegram message
  search rather than a generic Telegram operation.
