# Telegram Succinct Search Plain Text

- The consolidated `Telegram` internal tool now returns plain text for
  `search_messages` when `succinct` or `succint` is true.
- The subscriber still emits structured payloads internally, but the internal
  action formats them into a compact transcript before returning to the model.
- The text output includes the chat, query, returned match count, and context
  lines using relative offsets such as `+2 Sender: message`.
- Full JSON output remains the default when succinct mode is not requested.
