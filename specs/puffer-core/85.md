# Workflow Draft Command Hints

## Scope

The `/workflows` dashboard now advertises deterministic draft creation commands
beside connector trigger candidates.

## Behavior

- Trigger-ready connection rows include
  `draft=/workflows new <connection>-workflow <connection>`.
- Trigger-capable connector rows include the same draft command. Existing
  connections for that connector are preferred; otherwise the connector's
  suggested deterministic connection slug is used as a planned trigger.
- `draft`, `new`, `workflow`, and the rendered `/workflows new ...` command are
  searchable terms in connector and connection dashboard views.
- Setup-only connectors and non-trigger connections do not render draft
  commands.

## Compatibility

The change is additive to terminal output. Existing `repair=/connect ...`,
`monitor=/monitor ...`, connector capability labels, and `/workflows new`
creation semantics are unchanged.
