# Serve Connector Workflow Catalog UX

## Summary

The `/workflows connectors` view now exposes the serve-mode Discord, Matrix,
and webhook connector templates through the shared connector catalog.

## Behavior

- Searches such as `/workflows connectors discord` show the corresponding
  connector and deterministic `/connect` command hint.
- These connectors do not receive the `trigger` capability because they do not
  have workflow trigger readiness yet.
- Default `/workflows` summaries remain limited to trigger-ready connectors.

## Compatibility

The update is additive. Existing connector search, command hint, and trigger
readiness behavior is unchanged for current workflow-capable connectors.
