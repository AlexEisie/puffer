# Bitbucket Webhook Connect UX

`/connect bitbucket-webhook <connection-name>` now runs through the
deterministic serve-mode webhook preset flow.

## Contract

- The connector can be found by exact slug or searchable terms such as
  `bitbucket pull request`.
- The default deterministic command is
  `/connect bitbucket-webhook bitbucket-webhook`.
- Setup asks for bind address and URL path, defaulting blank paths to
  `/bitbucket`.
- The generated workspace config writes `[connectors.webhook]` with
  `welcome_message = "Bitbucket webhook ready."`.
- The `/workflows connectors` catalog row reports `runtime=serve` and
  `[no-trigger]`.

## Compatibility

The preset is setup-only. It does not claim connector protocol command support,
workflow trigger readiness, proxy routing, or outbound actions.
