# RequestSecret Request Metadata

`RequestSecret action="request"` now returns non-secret metadata alongside the
masked secret placeholder.

- Request output includes `username` and `origin` when they exist on the stored
  secret record.
- The raw secret value remains hidden behind a `PUFFER_SECRET_...` placeholder.
- Agents can fill email, phone, username, or account fields from `username`
  metadata while reserving the placeholder for password, token, API key, or
  other secret-value fields.
