# Workflow Connector Runtime Search

## Summary

`/workflows connectors` and `/connect` connector selection now expose connector
runtime/source hints in search text and rendered rows.

## Behavior

- `/workflows connectors` prints `runtime=<hints>` for each connector row.
- `/workflows connectors <query>` matches runtime hint terms such as `serve`,
  `subscriber`, `internal-tool`, and `command`.
- `/connect` searchable connector descriptions include runtime hints when the
  catalog provides them, and built-in fallback descriptions derive static hints.

## Compatibility

The command output is additive. Existing connector capability labels, trigger
status, action summaries, connection summaries, and deterministic connect
commands remain unchanged.
