# Telegram Succinct Search Internal Action

- The consolidated `Telegram` internal tool `search_messages` action now
  accepts `succinct` and the misspelled alias `succint`.
- The action forwards the compact-output preference to the
  `telegram-user` subscriber.
- Successful succinct responses tell the agent to inspect `results[].match`
  and `results[].context`, whose offsets are relative to the matched message.
