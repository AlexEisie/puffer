# Trello Webhook Connect UX

## Summary

Adds `trello-webhook` to the deterministic `/connect` and connector search UX.
Users can search for Trello board/card terms in the TUI catalog and run:

```text
/connect trello-webhook trello-webhook
```

## Design

The existing shared `connect_webhook_preset` dispatcher maps `trello-webhook`
to Trello display text, default path, and summary method.

Trello uses:

- bind prompt: `What bind address should the Trello webhook listen on?`
- path prompt: `What URL path should Trello post webhook events to?`
- default path: `/trello`
- welcome message: `Trello webhook ready.`
- method: `Trello event webhook`

## Compatibility

Existing webhook preset behavior is preserved for Asana, GitHub, GitLab, Jira,
Linear, and Stripe. The connector remains setup-only in `/workflows connectors`
and is searchable by Trello, board, card, and webhook terms.
