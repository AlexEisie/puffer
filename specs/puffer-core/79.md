# Workflow Connector Setup Search

## Summary

`/workflows connectors` now treats the visible setup command as searchable
connector metadata.

## Behavior

- Connector search includes the deterministic `/connect <connector_slug>
  <suggested_connection_slug>` command.
- Connector search includes the plain keywords `connect` and `setup`.
- The rendered connector row still shows the same `connect=/connect ...`
  command, now produced through the shared command formatter.
- Existing slug, description, skill, action, suggested connection, and existing
  connection-name search behavior is preserved.

## Compatibility

This is a read-only search improvement for the local slash-command dashboard. It
does not change workflow storage, connector setup commands, or daemon snapshot
contracts.
