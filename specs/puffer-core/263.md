# RelayDance Prompt-Only Video Capability Coverage

Date: 2026-06-09

## Summary

Core media tests now cover the expanded RelayDance static catalog and the
prompt-only request shape used by models without verified parameter metadata.

## Behavior

- Static video capability tests expect the expanded RelayDance model set.
- Grok Imagine Video is asserted as prompt-only in capability output.
- The RelayDance adapter builds prompt-only requests with only `model`,
  `prompt`, and `n` when a descriptor declares no parameters.

## Compatibility

The `relaydance_video` adapter wire contract is unchanged. Parameterized
Seedance descriptors still serialize `seconds` and `metadata.*` fields exactly
as before.
