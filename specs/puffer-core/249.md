# RequestSecret Search Hints

`RequestSecret action="search"` now searches with all non-secret metadata hints
provided by the agent.

- Search combines `query`, `label`, `description`, `username`, and `origin`.
- Runtime ignores empty hint fields before constructing the vault query.
- Origin-only or generic-query-plus-origin login searches can find imported
  site credentials.
- Search output still returns metadata only and does not include raw secret
  material or `PUFFER_SECRET_...` placeholders.
