# 56 - WorkflowCreate Trigger Support Validation

`WorkflowCreate` now rejects connections whose connector cannot produce inbound
events.

- A workflow trigger is allowed when the connector template has
  `can_subscribe=true`.
- Legacy subscriber-backed topics are also allowed when an installed manifest
  exists for either the connection slug or connector slug.
- Action-only connectors such as `lark-login`, `lark-app`, `slack-login`, and
  `slack-app` fail at creation instead of creating workflows that can never
  fire.

This keeps the workflow contract honest while still supporting static
subscriber topics such as email.
