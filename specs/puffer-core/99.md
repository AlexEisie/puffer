# Grafana `/connect` Webhook Preset

## Change

`/connect grafana-webhook <connection-name>` now runs the deterministic
serve-mode webhook preset flow.

## Flow

- Resolve `grafana-webhook` from exact slug or searchable terms such as
  `grafana alert`.
- Ask for the bind address.
- Ask for the URL path and default blank answers to `/grafana`.
- Write `[connectors.webhook]` in `.puffer/connectors.toml` with the selected
  display name, bind address, path, and `Grafana webhook ready.` welcome
  message.
- Return the same `puffer serve` next-step summary as the other serve-backed
  webhook presets.

## Compatibility

The preset remains setup-only. It does not claim workflow trigger support until
the serve-backed webhook runtime can publish connector trigger events directly.
