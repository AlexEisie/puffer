# Telegram Auth Probe Command

- `SubscriberCommand` now includes `telegram_auth_ok` for Telegram
  authorization probes that do not inspect peers, dialogs, messages, or update
  state.
- The command is control-plane only and returns status through subscriber
  events, matching the existing stdin/stdout subscriber command contract.
- This lets connection health checks avoid side effects that could consume
  pending Telegram updates before workflow routing sees them.
