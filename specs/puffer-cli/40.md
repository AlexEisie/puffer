# puffer-cli update 40

Adds the internal CLI surface for local Telegram Desktop auth import.

Design and behavior:

- `puffer internal-tool telegram import-local` sends `action=import_local` to
  the parent runtime.
- Optional flags: `--path`, `--account-index`, `--key-file`,
  `--passcode`, and `--passcode-stdin`.
- The shell helper alias remains `telegram`, so agents can run
  `telegram import-local` from the Telegram skill.

Security:

- `--passcode-stdin` is preferred for local passcodes.
- The passcode is only forwarded to the parent internal tool request and is
  not printed by the CLI adapter.
