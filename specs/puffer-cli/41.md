# Telegram QR And Desktop Commands

- `puffer internal-tool telegram import-desktop` is the preferred CLI spelling
  for importing Telegram Desktop `tdata`.
- `puffer internal-tool telegram import-local` remains a Clap alias for
  `import-desktop` and sends `action=import_desktop`.
- `puffer internal-tool telegram login-qr` sends `action=login_qr` with
  optional Telegram API credentials.
- `puffer internal-tool telegram login-qr-wait` sends `action=login_qr_wait`
  with an optional `--timeout-seconds` value.
- Phone-code login commands are unchanged and remain the fallback path.
