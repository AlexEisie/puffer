# Gmail Browser Connector Configuration Ownership

- The daemon no longer exposes `gmail_browser_update_config`; Gmail Browser configuration is created through `/connect gmail-browser`.
- Desktop settings snapshots no longer include a special `gmailBrowser` block. The daemon still reports Chrome profiles for general browser-session selection.
- The Gmail subscriber continues to read per-connection config from `${PUFFER_HOME}/gmail-browser-accounts/<connection>/config.toml`.
- Gmail connection auth checks still treat a connection as usable only when its per-connection config exists and contains at least one account.
