# BrowserAction DOM Inspect Runtime

`BrowserAction action="domInspect"` is now implemented by the daemon browser
runtime.

- The action accepts `query` as a CSS selector and returns bounded metadata for
  up to 25 matching elements.
- Returned element records include tag, role, accessible-ish name, bounded text,
  visibility, viewport rectangle, and selected non-secret attributes.
- The implementation does not return input values, so password and form content
  are not echoed through DOM inspection.
- Invalid or empty selectors fail the action instead of silently falling back to
  snapshots.
