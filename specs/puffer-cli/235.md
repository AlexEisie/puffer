# Monitor Task Next-Step Wording

- Monitor triage prompts now tell agents to write task subjects as next-step
  titles for the user, not neutral summaries of connector messages.
- Monitor task descriptions must explain why the item needs attention, preserve
  relevant source context and exact values, and state the concrete next step.
- Suggested action names and prompts should carry a concrete user choice forward
  so task rows help users decide what to do without opening the full source
  stream.
