# Desktop Timeline Generated Media Fallback

## Scope

The CLI desktop API timeline synthesizes chat attachments from
`ImageGeneration` tool output.

## Change

- Generated image attachments now call the shared core metadata fallback helper.
- If an artifact sidecar is missing, timeline attachments remain visible with
  `state = "missing"` using the MIME type and size from the tool output.
- Existing sidecar metadata remains authoritative when available.

## Compatibility

The timeline no longer drops generated image attachments solely because the
artifact sidecar is missing. Invalid ids, corrupt sidecars, and unsafe sidecar
paths still suppress the generated attachment.
