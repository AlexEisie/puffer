# Telegram Message Search CLI

- The internal Telegram CLI now exposes `telegram search-messages`.
- The command requires a message query and `--peer`, where `--peer` is a
  numeric id returned by `telegram search-peers` or a public `@username`.
- Optional `--limit` bounds match count and optional `--context` controls how
  many surrounding messages are returned before and after each match.
- The CLI forwards the request to the parent runtime as the consolidated
  Telegram internal tool action `search_messages`.
