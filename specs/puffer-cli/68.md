# Workflow Connection Connect Commands

## Summary

The desktop daemon workflow snapshot now carries deterministic connection repair
commands for each stored connector connection.

## Behavior

- Connection rows include `connect_command`.
- The command is formatted as `/connect <connector_slug> <connection_slug>`.
- The command is emitted for every connection, including connections that
  cannot currently trigger workflows.
- `monitor_command` remains limited to trigger-ready event connections.

## Compatibility

The field is additive. Older desktop clients can ignore it, while newer clients
can run connector auth repair without reconstructing command strings.
