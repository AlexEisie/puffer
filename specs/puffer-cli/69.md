# Workflow Snapshot Connector Runtime Hints

## Summary

The desktop daemon workflow snapshot now includes `runtime_hints` on each
connector catalog row.

## Contract

- `workflow_list` connector rows include `runtime_hints: string[]`.
- Existing fields such as `connector_slug`, `connect_command`,
  `can_trigger_workflow`, and `action_slugs` are unchanged.
- The hints are produced by `puffer-subscriptions` so Corbina and core command
  surfaces use the same source classification.

## Compatibility

Older clients can ignore `runtime_hints`. New clients should treat it as
optional because stale daemons or test fixtures may omit it.
