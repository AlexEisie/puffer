# Pipeline Monitor Commands

## Summary

Corbina's Pipelines connector picker can now start connector task monitors from
the GUI.

## Behavior

- Event-capable connection rows show a monitor action beside the workflow
  trigger action.
- Selecting the monitor action starts `/monitor <connection>` through the same
  agent-session command path used for deterministic connector setup commands.
- The action is disabled for connections that cannot start workflow triggers,
  while connector setup rows remain selectable for `/connect` staging.

## Compatibility

The workflow trigger picker keeps its existing row labels, selection behavior,
and connector setup command behavior. The monitor button is additive and does
not alter draft pipeline state.
