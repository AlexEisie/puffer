# Borderless Media Settings Loading

## Scope

Image and video generation settings loading states should stay visible without
adding bordered modal chrome.

## Change

- The shared media settings loading block no longer draws an outer border.
- Image and video loading states still share the same spinner, spacing, role,
  and media-specific loading copy.

## Compatibility

This is a presentation-only change. Capability loading, provider selection, and
saved media settings behavior are unchanged.
