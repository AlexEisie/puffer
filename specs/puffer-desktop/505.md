# Asana Connector Catalog GUI

## Summary

Adds the Asana webhook preset to the Corbina/Pipelines connector catalog test
fixture and search coverage.

## UI Contract

The catalog row exposes:

- slug: `asana-webhook`
- description: `Asana task, project, and story webhook preset backed by puffer serve`
- skill: `asana-webhook`
- runtime hint: `serve`
- trigger state: setup-only / no trigger
- command: `/connect asana-webhook asana-webhook`

Search terms such as `asana task project webhook`, `serve webhook`, `serve`,
`no trigger`, and `setup-only webhook` include the Asana row. Selecting it shows
the same details and runnable deterministic connect command as the other
serve-backed webhook presets.

## Compatibility

Connector catalog result totals increase from 16 to 17. Serve-backed connector
counts increase from 9 to 10, and setup-only webhook counts increase from 6 to
7. Existing connection and workflow draft UX is unchanged.
