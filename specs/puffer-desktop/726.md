# Media Settings Loading Parity

## Scope

Image and video generation settings must not render an empty modal body while
their media capability request or settings snapshot is still pending.

## Change

- The media settings modal renders one shared loading block before content is
  ready.
- Loading copy remains media-specific: image settings mention image generation
  models, and video settings mention video generation models.
- The same spinner, role, sizing, and muted panel styling are used for both
  media kinds.

## Compatibility

The capability request shape, saved media settings shape, provider selection,
and video parameter normalization are unchanged. This only restores a visible
pending state so slower video discovery does not look blank.
