# 756 - Gemini Provider Icon

## Summary

Google provider cards now use the official Gemini mark instead of the generic
Google "G" logo.

## Design

- The existing `/service-icons/google.svg` path is preserved so provider visual
  mapping and runtime asset lookup do not change.
- The icon artwork is replaced with the Gemini star mark from Google's Gemini
  logo, keeping the provider id and accent behavior unchanged.
- The checked-in `dist/service-icons/google.svg` copy is kept in sync with the
  public asset.

## Compatibility

No provider ids, auth flows, model selection behavior, or desktop API contracts
change. Existing Google provider references continue to resolve the same icon
path, now rendering the Gemini artwork.
