# Selectable Connector Setup Rows

## Summary

The Pipelines connector catalog now lets users select connector templates even
when those connectors cannot start workflow triggers.

## Behavior

- Non-trigger connector template rows remain searchable and visible.
- Selecting a non-trigger connector row does not mutate the workflow trigger.
- The selected row still exposes its deterministic setup command so the user can
  copy or inspect the next connector setup step.
- Existing connection rows that cannot start workflow triggers remain disabled.

## Compatibility

Trigger-ready connector rows keep the existing behavior of staging a planned
connection trigger. Non-trigger connector rows only affect the selected catalog
row and setup command display.
