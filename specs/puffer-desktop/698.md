# puffer-desktop Update 671: Media Settings Menu Icons

## Summary

Composer media generation settings actions now use media-specific Add content
menu icons.

## Frontend Contract

- `Image generation settings` uses the Lucide `Image` icon through the shared
  desktop `Icon` wrapper.
- `Video generation settings` uses the Lucide `Video` icon through the shared
  desktop `Icon` wrapper.
- Existing menu labels, click handlers, modal behavior, and media settings
  payload contracts are unchanged.

## Tests

- Svelte type checking covers the new shared icon names and imports.

## Compatibility Notes

No daemon or persisted settings contract changes are required.
