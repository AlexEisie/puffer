# 759 - Thinking Loading Dots

## Summary

The desktop session view now shows animated three-dot loading feedback while an
agent turn is in the default thinking state.

## Design

- `LoadingDots.svelte` provides a reusable Svelte/CSS loading indicator with
  three `#c1c1c1` 6px dots, 4px spacing, staggered vertical motion, glow, and
  reduced-motion support.
- `ConversationView.svelte` uses the loader only when the active turn is
  running, thinking, and has no custom status hint beyond `Thinking`.
- The elapsed timer remains visible beside the dots so long-running turns still
  expose timing context.
- Non-thinking status hints continue to render as text, preserving existing
  tool, permission, and waiting state behavior.

## Compatibility

The change avoids adding React-only animation dependencies to the Svelte desktop
app. Existing text statuses remain unchanged, and users with reduced-motion
preferences receive a static three-dot indicator.
