# Gmail Browser Settings

- Adds Gmail Browser connector configuration to the Settings snapshot.
- Browser profile rows now include Google accounts discovered from Chrome profile metadata.
- General settings includes a Gmail Browser configuration control with Chrome profile selection, Gmail account multi-select, and a save action that persists connector config through `gmail_browser_update_config`.
- Saving Gmail Browser settings creates or refreshes the default `gmail-browser` connection so Task/Monitor workflows can subscribe to browser-streamed Gmail messages.
