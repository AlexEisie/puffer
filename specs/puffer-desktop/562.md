# Task Monitor Ignore Controls

The Tasks screen now exposes monitor-task ignore controls.

- Monitor rows show a single `Ignore` control while they are not already
  ignored.
- When a monitor task provides `possibleIgnoreReasons`, the `Ignore` control
  opens a compact dropdown of ignore choices; choosing an item immediately
  marks the task ignored.
- Ignoring a task calls the daemon ignore RPC, refreshes the task snapshot, and
  leaves the user on the Tasks screen.
- Ignored tasks move out of the All/Agent/Monitor views into a dedicated
  `Ignored` tab and use an `ignored` status label there.
