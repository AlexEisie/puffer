# Pipelines Connector Connection Names

## Summary

The Pipelines connector catalog now shows concrete connection names on connector
rows and indexes those names for search.

## Behavior

- Connector rows show up to two existing connections as `conn:<slug>` tags.
- Additional connections are summarized as `+N conn`.
- Connector search indexes existing connection slugs and descriptions for each
  connector.
- Searching by a connection name, such as `telegram-user`, keeps the owning
  connector row visible alongside the concrete connection row.
- Searching by connection description terms, such as `workspace slack`, also
  keeps the owning connector row visible.

## Compatibility

This change only affects Pipelines search metadata and visible connector row
tags. It does not change workflow snapshots, connection records, connector
commands, or persisted desktop state.
