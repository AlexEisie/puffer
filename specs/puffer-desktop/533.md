# Connection Append Query Intent

The Corbina Pipelines connection list now lets command-like connector searches
parameterize a connection append action.

- When the connector search names a connection or connector and includes an
  explicit append path, the connection row's append button uses that path.
- Remaining non-routing words become the append filter pattern, so searches like
  `append telegram-user hi /tmp/hi` create a binding for matching `hi` events
  written to `/tmp/hi`.
- Broad append searches that do not name a connection keep the previous default
  connection append command, avoiding noisy matches across every connection.
