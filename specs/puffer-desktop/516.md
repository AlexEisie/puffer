# Azure DevOps Pipeline Connector Catalog

Corbina's Pipelines connector catalog now includes the
`azure-devops-webhook` setup-only row in fake daemon fixtures and Playwright
coverage.

## Behavior

- The all-connectors count is `28/28`.
- Serve and setup-only webhook filters include `azure-devops-webhook`.
- Searching `azure devops work item webhook` surfaces the Azure DevOps setup
  row.
- Selecting the row shows `skill:azure-devops-webhook`, the serve runtime
  hint, and the deterministic command
  `/connect azure-devops-webhook azure-devops-webhook`.

## Compatibility

Azure DevOps remains a setup-only connector in the UI: the row is selectable
for connection setup, but it does not present itself as workflow-trigger ready.
