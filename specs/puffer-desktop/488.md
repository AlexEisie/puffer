# Pipelines Connector Runtime Hints

## Summary

Corbina Pipelines now renders and searches connector runtime/source hints from
the workflow snapshot.

## Behavior

- `WorkflowConnector.runtime_hints` is optional for backward compatibility.
- Connector catalog rows render each runtime hint beside existing capability
  tags such as `auth`, `events`, `trigger`, and action slugs.
- Connection rows inherit their connector runtime hints so configured
  connections can also be found by source terms.
- Connector search indexes runtime hints for both connector and connection
  results.

## Compatibility

No existing workflow, connection, or connector command shape changes. Missing
runtime hints are treated as an empty list.
