# Pipeline Connector Row Setup Actions

## Summary

Corbina Pipelines can start default connector setup directly from connector
catalog rows.

## Behavior

- Each connector catalog row has a plug action that runs the deterministic
  default `/connect <connector_slug> <suggested_connection_slug>` command.
- Connector search indexes `connect`, `setup`, `connect_command`, and the
  suggested connection slug so setup-oriented queries find the right row.
- Row actions use the same default command shown by the connector setup panel.
- Setup, repair, and monitor command buttons continue to share one busy guard
  so only one agent-turn command launches at a time.
- The Pipelines fixture catalog now includes Lark app and user-token connectors
  so GUI coverage exercises another real connector family.

## Compatibility

This is additive UI behavior. Existing connector row selection, workflow trigger
staging, and selected setup-panel commands keep their current behavior.
