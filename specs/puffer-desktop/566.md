# Paper sidebar parity

- The desktop sidebar now follows the latest Paper `Sidebar` frame: a flat `#fafafa` rail, 12px outer padding, 34px nav rows, lavender active state, and the compact account chip.
- The visible agent area is now a reusable `SidebarProjects` component with a `Projects` list, library-backed chevron/folder icons, and visible per-agent section rows below expanded projects. The previous active-agent state filter is not part of the Paper sidebar treatment.
- Existing sidebar APIs for active agents, project collapse state, pinning, opening, collapse toggling, and resizing remain wired so workspace data and callbacks stay compatible with the shell.
- All sidebar icons continue to render through `Icon.svelte`, which maps project icon names to `lucide-svelte`; no inline or hand-drawn SVG icons were added.
