# Task Configuration Popup

The Tasks screen now uses a configuration popup for monitor setup and memory
editing.

- The top Tasks action opens a `Task configuration` dialog instead of showing
  an inline new-monitor form.
- The dialog includes the new-monitor connection picker and create action.
- Expanded monitor memory panels expose `Edit`, which opens the same dialog
  with that memory selected.
- The dialog edits monitor memory in a monospace textarea with Reset and Save
  controls.
- Save calls the daemon monitor-memory save RPC and refreshes the task
  snapshot on success.
- Truncated memory snapshots are view-only to avoid overwriting unseen content.

The memory section remains collapsed by default so task rows stay scannable.
