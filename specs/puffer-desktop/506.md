# Trello Connector Catalog GUI

## Summary

Adds the Trello webhook preset to the Corbina/Pipelines connector catalog test
fixture and search coverage.

## UI Contract

The catalog row exposes:

- slug: `trello-webhook`
- description: `Trello board, card, list, and comment webhook preset backed by puffer serve`
- skill: `trello-webhook`
- runtime hint: `serve`
- trigger state: setup-only / no trigger
- command: `/connect trello-webhook trello-webhook`

Search terms such as `trello board card webhook`, `serve webhook`, `serve`,
`no trigger`, and `setup-only webhook` include the Trello row. Selecting it
shows the same details and runnable deterministic connect command as the other
serve-backed webhook presets.

## Compatibility

Connector catalog result totals increase from 17 to 18. Serve-backed connector
counts increase from 10 to 11, and setup-only webhook counts increase from 7 to
8. Existing connection and workflow draft UX is unchanged.
