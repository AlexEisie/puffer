# Tauri Timeline Generated Media Fallback

## Scope

The Tauri desktop backend timeline synthesizes chat attachments from
`ImageGeneration` tool output.

## Change

- Generated image attachments now use the shared core metadata fallback helper.
- If an artifact sidecar is missing, the assistant timeline keeps the generated
  image attachment with `state = "missing"`.
- The fallback MIME type and byte count come from the tool output artifact.

## Compatibility

The Tauri timeline behavior now matches the CLI desktop API. Existing sidecars
remain authoritative, and unsafe or corrupt sidecars do not fall back.
