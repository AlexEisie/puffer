# Corbina GitHub Webhook Connector

## Summary

The Pipelines connector picker surfaces the `github-webhook` preset from the
daemon connector catalog.

## Contract

- The fake daemon fixture includes `github-webhook` with `serve`, `no trigger`,
  `no auth`, and `/connect github-webhook github-webhook` metadata.
- Connector search matches GitHub-specific terms such as `github event webhook`
  and generic terms such as `serve webhook`.
- The setup-only row can run the deterministic connect command from the picker.
- Result counts account for the twelfth built-in connector while trigger,
  draft, action, monitor, and repair filters keep their existing semantics.

## Compatibility

No new Pipelines protocol fields are required. The UI continues to render
connector metadata returned by `ConnectorList`, so the real daemon catalog and
the test fixture share the same shape.
