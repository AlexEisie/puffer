# Connector search connection names

- The Pipelines connector catalog keeps normal all-term search matching first,
  preserving descriptive searches such as `telegram personal`.
- When no connector matches every term and the final query token is a valid
  connection slug, the catalog falls back to the first connector-search token
  after generic setup words and treats the final token as the staged connection
  name.
- This lets searches such as `connect matrix matrix-main` and `email team-email`
  select the connector and stage deterministic `/connect <connector>
  <connection>` commands without manually editing the connection field.
