# Serve Connector Picker Visibility

## Summary

The desktop Pipelines connector picker can show serve-mode connector templates
such as Discord while preserving workflow trigger readiness safeguards.

## Behavior

- Serve-mode connector rows are searchable in the connector catalog.
- Rows without workflow-trigger readiness are disabled and labeled `no trigger`.
- Selecting unavailable serve-mode connector rows cannot mutate the workflow
  draft trigger.

## Compatibility

This follows the existing readiness metadata contract. No daemon schema changes
are required beyond the catalog rows already supplied in the workflow snapshot.
