# Shell Sidebar Visual Refresh

## Change

The desktop shell sidebar has been visually tightened. The top rail now has a
proper Puffer Code brand row, navigation items use softer active and hover
states, the agents controls are cleaner, empty agent results are contained, and
the account footer is styled as a compact card.

## Architecture

- `Sidebar.svelte` adds semantic brand copy and replaces inline account avatar
  styling with a reusable sidebar avatar class.
- `app.css` owns the sidebar presentation, including background, section
  separators, nav item states, filter styling, empty state, and account footer.
- Collapsed sidebar behavior keeps the existing icon-only layout and hides the
  new brand copy.

## Compatibility

This is a presentation-only shell change. Screen routing, agent grouping,
filtering, resize behavior, account routing, and workflow screen state are
unchanged.
