# Vercel Connector Catalog Row

## Scope

Adds `vercel-webhook` to Corbina's Pipelines connector catalog fixture and
search coverage.

## Design

- The fake daemon exposes `vercel-webhook` with `serve` runtime hints, no
  trigger support, no actions, and `/connect vercel-webhook vercel-webhook`.
- Pipelines UI tests verify the connector appears in all connector coverage,
  serve-filter, setup-only webhook-filter, and selected-connector detail flows.
- Result count expectations increase from 29 to 30 total connectors and from
  19 to 20 setup-only webhook results.

## Compatibility

This is test fixture coverage for the desktop GUI. Runtime behavior still comes
from the daemon connector catalog returned by Puffer.
