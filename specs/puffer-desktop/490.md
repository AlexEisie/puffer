# Pipelines Monitor Filter Preset

## Summary

Corbina Pipelines now includes a `Monitor` connector-search preset for finding
connections that can start connector task monitors.

## Behavior

- The `Monitor` preset writes `monitor` into the existing connector search box.
- Monitorable connections index both `monitor` and `monitorable`, in addition
  to their backend-provided monitor command.
- Connection search now includes connection lifecycle state and consumer state,
  so future presets or manual searches can target `active`, `idle`, and exact
  state names without a second filtering model.

## Compatibility

The preset reuses the existing all-terms substring matcher and result summary.
No connector setup or workflow trigger selection behavior changes.
