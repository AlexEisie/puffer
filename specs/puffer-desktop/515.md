# Bitbucket Pipeline Connector Catalog

Corbina's Pipelines connector catalog now includes the `bitbucket-webhook`
setup-only row in fake daemon fixtures and Playwright coverage.

## Behavior

- The all-connectors count is `27/27`.
- Serve and setup-only webhook filters include `bitbucket-webhook`.
- Searching `bitbucket pull request webhook` surfaces the Bitbucket setup row.
- Selecting the row shows `skill:bitbucket-webhook`, the serve runtime hint, and
  the deterministic command `/connect bitbucket-webhook bitbucket-webhook`.

## Compatibility

Bitbucket remains a setup-only connector in the UI: the row is selectable for
connection setup, but it does not present itself as workflow-trigger ready.
