# puffer-desktop Update 672: Media Settings Modal Alignment

## Summary

Image and video generation settings now use the same compact modal visual
language as the workspace `Choose provider` dialog.

## Frontend Contract

- `MediaSettingsModal` keeps its existing dialog names, form labels, loading
  states, save labels, and media config payload behavior.
- The modal shell now matches the provider chooser rhythm: centered scrim,
  rounded card, compact title bar, 28px close control, and shared button
  styling.
- Provider and model controls render as visible labelled `select` controls.
- All media setting selects render as a single vertical column.
- Image size options include common square, portrait, landscape, and high
  resolution presets in addition to provider-advertised values.
- Image settings use `Save` as the primary action label. Video settings keep the
  existing `Save video generation settings` label.

## Tests

- Svelte type checking covers the updated modal structure and styles.
- Existing Playwright media settings tests continue to target the same dialog
  labels and select labels.

## Compatibility Notes

No daemon API, settings persistence, or capability discovery contract changes
are required.
