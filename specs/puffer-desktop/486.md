# Pipelines Setup-Only Connector Search

## Summary

The Pipelines connector picker search now matches the same setup-only capability
terms used by the TUI workflow connector catalog.

## Behavior

- Connector rows index visible capability tags in addition to slug, description,
  skill, connect command, suggested connection name, setup keywords, and action
  slugs.
- Trigger-capable connectors index `trigger` and `trigger-ready`.
- Setup-only connectors index `no trigger`, `no-trigger`, and `setup-only`.
- Connection rows index `trigger`/`trigger-ready` or
  `no trigger`/`no-trigger`/`setup-only` based on whether the connection can
  start workflow triggers.

## UX

Users can search `no trigger` to find connector setup rows that can be
configured but cannot directly start workflow triggers. Combining capability
terms with connector names, such as `setup-only webhook`, narrows the catalog to
the intended connector.

## Compatibility

This is a search metadata change only. It does not alter workflow snapshots,
connector APIs, connection records, or persisted desktop state.
