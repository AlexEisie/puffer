# Session detail header cleanup

- Session detail headers no longer show the purple Puffer state icon beside the title.
- The project folder name is removed from the header metadata row so the title area only shows the session name/title and edit affordance.
- Branch and worktree metadata remain supported when present, preserving useful repository context without showing the current project folder label.
- The rest of the detail header behavior is unchanged, including back navigation, title rename, status pill, tabs, and close action.
