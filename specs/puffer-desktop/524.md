# Pipelines action discoverability

Corbina's Pipelines search now keeps workflow action cleanup and connector
capability discovery aligned with the terminal workflow dashboard.

## UX

- Workflow binding rows are searchable by `delete`, `remove`, `cleanup`, and
  the full `/workflows delete <binding-slug>` command text.
- Connector dashboard filtering computes query terms once and reuses them
  across connector, connection, monitor, action, and monitor-task filters.
- Connector catalog rows stay compact and still show at most three matching
  action chips plus a hidden count.
- The selected connector detail panel shows every action slug so users can
  inspect a connector's full action surface before deciding how to wire it into
  workflows.

This change is UI-only. It does not add connector templates, change connector
auth behavior, or alter workflow binding storage.
