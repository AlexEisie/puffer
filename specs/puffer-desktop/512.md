# Datadog Connector Catalog UI

## Change

The desktop Pipelines connector catalog fixture now includes
`datadog-webhook` with the same setup-only serve metadata returned by the
daemon catalog.

## UI Contract

- The all-connectors count is 24 connectors.
- Search terms such as `serve webhook`, `serve`, `setup-only webhook`, and
  `datadog monitor webhook` surface the Datadog setup row.
- The setup row shows `serve` and `no trigger` status metadata.
- The selected connector detail pane exposes `skill:datadog-webhook` and the
  deterministic command `/connect datadog-webhook datadog-webhook`.

## Compatibility

This changes only the test daemon fixture and Pipelines regression coverage.
Production desktop rendering uses the daemon connector catalog payload and does
not require connector-specific UI branching.
