# Alertmanager Connector Catalog UI

## Change

The desktop Pipelines connector catalog fixture now includes
`alertmanager-webhook` with the same setup-only serve metadata returned by the
daemon catalog.

## UI Contract

- The all-connectors count is 23 connectors.
- Search terms such as `serve webhook`, `serve`, `setup-only webhook`, and
  `prometheus alertmanager webhook` surface the Alertmanager setup row.
- The setup row shows `serve` and `no trigger` status metadata.
- The selected connector detail pane exposes `skill:alertmanager-webhook` and
  the deterministic command `/connect alertmanager-webhook alertmanager-webhook`.

## Compatibility

This changes only the test daemon fixture and Pipelines regression coverage.
Production desktop rendering uses the daemon connector catalog payload and does
not require connector-specific UI branching.
