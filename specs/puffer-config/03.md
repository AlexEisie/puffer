# puffer-config update 03

Updates environment-variable inventory for Telegram Desktop import.

Design and behavior:

- `HOME` now records the Telegram subscriber import path resolver, which uses
  it for `~` expansion and default Linux/macOS Telegram Desktop paths.
- `LOCALAPPDATA` now records the Windows Telegram Desktop default tdata path
  lookup in addition to Chrome discovery.

Compatibility:

- Runtime config discovery is unchanged; this is documentation inventory only.
