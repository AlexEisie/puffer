# Telegram Message Search Media Values

- Telegram message-search payloads now represent media as a single string
  value.
- Downloadable media, including photos, documents, and stickers, is saved
  under the subscriber state directory and surfaced as a local file path.
- Semantic media, including polls, contacts, locations, dice, venues, live
  locations, and web pages, is surfaced as concise text.
- Full JSON output keeps `text` as the Telegram caption/text and adds `media`
  as the path/text string when the message has media.
- Succinct output merges the media string into the context line before the
  caption, so image-only context messages show as a file path instead of an
  empty line.
