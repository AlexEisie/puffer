# Telegram Media Path Or Text Contract

- Message-search media values now follow a path-or-text contract.
- Photos, documents, and stickers are downloaded to
  `telegram-media/<chat-id>/` under the subscriber state directory and exposed
  as local file path strings.
- Polls and other semantic media are exposed as text strings, for example
  `poll: Ship it? | Yes / No`.
- Search continues when a download fails; the media string becomes a concise
  unavailable-media text value instead of aborting the whole search.
