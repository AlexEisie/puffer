# Azure DevOps Connector Template

The built-in connector catalog now includes `azure-devops-webhook` as a
serve-mode webhook preset.

## Template Contract

- Slug and deterministic connection name: `azure-devops-webhook`.
- Description: Azure DevOps code and work item webhook preset backed by
  `puffer serve`.
- Skill: `azure-devops-webhook`.
- Binary: `puffer connector webhook`.
- Auth: no connector auth requirement at catalog time.
- Runtime: setup-only, served by `puffer serve`, no subscriber process, no
  workflow trigger capability, no proxy capability, and no outbound actions.
- Output schema: `azure_devops_event` with event, project, repository, actor,
  pull request, work item, branch, state, message, and URL fields.

## Compatibility

The generic `webhook` connector remains available for arbitrary HTTP
callbacks. `azure-devops-webhook` is only a named preset with better discovery,
schema, skill, and deterministic setup command metadata.
