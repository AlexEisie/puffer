# Asana Webhook Connector Preset

## Summary

Adds `asana-webhook` to the built-in connector catalog as a serve-backed
setup-only webhook preset. The connector is intended for Asana task, project,
story, and related resource event deliveries.

## Catalog Contract

- `connector_slug`: `asana-webhook`
- `suggested_connection_slug`: `asana-webhook`
- `skill`: `asana-webhook`
- `binary`: `puffer connector webhook`
- `requires_auth`: `false`
- `can_subscribe`: `false`
- `can_proxy_agent`: `false`
- `actions`: none
- `output_schema`: `asana_event_output_schema`

The preset does not claim connector protocol runtime support. It is surfaced so
`/workflows connectors`, `/connect`, and GUI connector pickers can guide users
to configure the generic serve-mode webhook connector with Asana-specific
defaults.

## Compatibility

The Asana preset follows the existing webhook preset model used by GitHub,
GitLab, Jira, Linear, and Stripe. It remains setup-only until the workflow
runtime supports serve-mode webhook triggers as first-class workflow triggers.
