# Vercel Webhook Catalog Metadata

## Scope

Adds `vercel-webhook` to the built-in workflow connector template catalog.

## Design

- The connector is a serve-mode webhook preset with no subscriber command, no
  workflow trigger runtime, no proxy-agent support, and no outbound actions.
- The output schema describes normalized Vercel event fields including event
  type, delivery id, deployment id, project id, domain, flag, target, actor,
  message, and URL.
- Suggested connection slug remains deterministic: `vercel-webhook`.

## Compatibility

The connector is setup-only and uses the same generic webhook serve contract as
the existing webhook presets.
