# Monitor task prefilter superseded

This spec recorded the first deterministic monitor task prefilter. It is no
longer active.

- The low-signal gate was removed in `76.md` after delayed digest triage became
  the monitor default.
- New monitor events are not skipped with `monitor_task_gate_skip`.
- Router-side skips remain limited to explicit routing policy such as muted
  notifications, ignore filters, contact filters, include filters, and
  classifier rejection.

The active design favors preserving direct-chat context for the delayed digest
agent over pruning casual messages before triage.
