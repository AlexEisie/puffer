# Google Calendar Monitor Rule Fields

## Summary

The `gcal-browser` monitor rule menu exposes only the stable, user-facing
calendar filters: event content and event title.

## Contract

- `Event content` continues to map to the normalized event text.
- `Event title` maps to `event.title`.
- `When`, `Location`, and `Event URL` are not selectable monitor rule fields.

## Compatibility

The browser poller still emits the raw agenda row under `payload.event`.
Existing persisted filters that target removed fields may remain in older
bindings, but new UI-created GCal filters only use the stable menu entries.
