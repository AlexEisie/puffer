# Bitbucket Connector Template

The built-in connector catalog now includes `bitbucket-webhook` as a serve-mode
webhook preset.

## Template Contract

- Slug and deterministic connection name: `bitbucket-webhook`.
- Description: Bitbucket push and pull request webhook preset backed by
  `puffer serve`.
- Skill: `bitbucket-webhook`.
- Binary: `puffer connector webhook`.
- Auth: no connector auth requirement at catalog time.
- Runtime: setup-only, served by `puffer serve`, no subscriber process, no
  workflow trigger capability, no proxy capability, and no outbound actions.
- Output schema: `bitbucket_event` with event, repository, actor, pull request,
  branch, commit, message, and URL fields.

## Compatibility

The generic `webhook` connector remains available for arbitrary HTTP callbacks.
`bitbucket-webhook` is only a named preset with better discovery, schema, skill,
and deterministic setup command metadata.
