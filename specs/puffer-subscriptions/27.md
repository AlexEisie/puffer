# GitHub Webhook Connector Preset

## Summary

The built-in connector catalog includes `github-webhook`, a GitHub event
webhook preset backed by the existing `puffer serve` webhook transport.

## Design

- The connector slug and suggested connection slug are both
  `github-webhook`.
- Runtime hints include `serve`; the connector has no subscriber manifest,
  command runtime, workflow trigger support, or connector actions.
- The output schema describes normalized GitHub event payloads with event,
  action, repository, sender, message, and optional URL fields.
- The preset advertises the `github-webhook` skill so setup guidance can stay
  resource-backed.

## Compatibility

This is additive to the connector catalog. The existing generic `webhook`
connector stays available and keeps its prior slug, schema, setup command, and
serve-mode runtime behavior.
