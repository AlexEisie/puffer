# Lark Connector Catalog

## Scope

Adds built-in `lark-app` and `lark-login` connector templates.

## Templates

- `lark-app`: custom app credentials over OpenAPI.
- `lark-login`: user access token over OpenAPI.

Both templates:

- use skill `lark`
- use binary label `puffer internal-tool lark`
- require auth
- do not claim subscription support
- do not claim agent proxy support
- expose `send_message`, `react`, `send_reaction`, and `remove_reaction`

## Permissions

`send_message` uses `external_message_send`. Reaction actions use
`external_message_interaction`. All actions are marked external side effects.

## Compatibility

The catalog does not add `lark-bot` until there is an event subscriber or bot
proxy implementation. This avoids enabling workflows that would require a
nonexistent `subscribe` runtime.
