# Trello Webhook Connector Preset

## Summary

Adds `trello-webhook` to the built-in connector catalog as a serve-backed
setup-only webhook preset. The connector is intended for Trello board, card,
list, checklist, and comment action deliveries.

## Catalog Contract

- `connector_slug`: `trello-webhook`
- `suggested_connection_slug`: `trello-webhook`
- `skill`: `trello-webhook`
- `binary`: `puffer connector webhook`
- `requires_auth`: `false`
- `can_subscribe`: `false`
- `can_proxy_agent`: `false`
- `actions`: none
- `output_schema`: `trello_event_output_schema`

The preset does not claim connector protocol runtime support. It is surfaced so
`/workflows connectors`, `/connect`, and GUI connector pickers can guide users
to configure the generic serve-mode webhook connector with Trello-specific
defaults.

## Compatibility

The Trello preset follows the existing webhook preset model used by Asana,
GitHub, GitLab, Jira, Linear, and Stripe. It remains setup-only until the
workflow runtime supports serve-mode webhook triggers as first-class workflow
triggers.
