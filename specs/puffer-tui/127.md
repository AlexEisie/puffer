# Workflow Monitor Task Popup Coverage

## Summary

Adds TUI popup regression coverage for discovering monitor task workflow views.

## Behavior Covered

- Typing `/workflows ta` renders the `/workflows tasks` popup row.
- The row describes the monitor task search and action surface.
- The hint remains visible so users can discover that the subcommand accepts a
  query.

## Compatibility

- This is test coverage only.
- No rendering, command dispatch, popup ranking, or workflow command behavior is
  changed.
