# Workflow Draft Hints In TUI

## Scope

TUI users can now discover connector-backed workflow draft creation from the
same `/workflows` views used for connector setup and monitor routing.

## Behavior

- `/workflows connections` shows a `draft=/workflows new ...` action for every
  trigger-ready connection.
- `/workflows connectors` shows a `draft=/workflows new ...` action for every
  trigger-capable connector, using an existing connection when present or the
  deterministic suggested connection slug for a planned trigger.
- Connector and connection filters include `draft`, so users can narrow the
  terminal dashboard to draft-capable rows.

## Compatibility

This only changes text rendered by the slash-command dashboard. The TUI overlay,
AskUserQuestion handling, workflow storage, and connector authentication flows
are unchanged.
