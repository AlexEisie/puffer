# Connect Search-First Names

The TUI `/connect` popup now matches Corbina's connector-search precedence:
search terms are honored before custom connection-name inference.

- Multi-term connector queries such as `/connect telegram personal` and
  `/connect vote poll` keep using all terms as connector catalog search terms
  and complete to the connector's suggested connection slug.
- Custom connection names are inferred only when no connector or configured
  connection matches all typed terms and the query can fall back to a connector
  search token, for example `/connect matrix matrix-main`.
- Exact deterministic `/connect <connector-slug> <connection-name>` input still
  hides the popup and submits directly through the core command path.

This change is TUI-only. It does not change connector auth, connection storage,
Corbina, workflow execution, or connector runtime behavior.
