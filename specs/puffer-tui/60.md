# Searchable User Questions

The TUI `AskUserQuestion` overlay now supports searchable single-select
choice questions.

- When a question has `searchable: true`, typed characters filter the option
  list instead of activating the `Other` custom-answer row.
- Arrow and page navigation move through the filtered result set, and Enter
  returns the selected option label.
- The composer placeholder and footer hint switch to search language for
  searchable questions.
- Non-searchable choice and input prompts retain their previous custom answer,
  shortcut, multi-select, and preview behavior.
