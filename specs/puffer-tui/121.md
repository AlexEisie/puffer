# Append Query Intent Completion

The TUI slash popup now mirrors Corbina's connection append search intent for
explicit `/workflows append ...` queries.

- `/workflows append <connection-or-connector> <pattern> <path>` suggestions use
  the provided append path and turn remaining non-routing words into the filter
  pattern.
- Generated completions preserve the planned connector `--connector` suffix when
  the selected connection is not already configured.
- Broad path-only append searches continue to return no rows instead of
  matching every trigger-capable connector.
