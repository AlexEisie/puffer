# puffer-tui update 54

Adds display handling for Telegram local auth import.

Design and behavior:

- `TelegramImportLocal` renders as `Telegram Import`.
- Headers show the import path when provided, otherwise `local Telegram
  Desktop`.
- Local passcodes are never summarized in tool headers.

Compatibility:

- Existing Telegram login code and 2FA redaction behavior is unchanged.
